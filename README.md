# 🎥 Netflix TV shows and Movies

## 📌 Project overview
This project performs a exploratory data analysis(EDA) on the datset Netflix TV shows and Movies dataset using Python. The objective of this project is to analyze the netflix's content and discover meaningful insights about the movies, TV shows, genres, ratings, release year and many more using data visualization.


## 📂 Dataset
**Source : **Kaggle
**File name : **netflix_titles.csv

### Dataset Description
The datset contains the meaningful information about the netflix content such as movies and TV shows. It includes details such as :

- show_id
- type
- title
- director
- cast
- country
- date_added
- release_year
- rating
- duration
- listed_in (Genre)
- description


## 🚀 Technologies used

* Python - Prgramming language
* Pandas
* Matplotlib
* Seaborn
* VS Code - Code editor

## 📊 Exploratory Data Analysis

The following analysis were performed during EDA:
- Movies VS TV shows distribution
- Ratings distribution
- Release year distribution
- Top 10 Countries by content
- Movie Duration
- Top 10 Directors
- Top Genres


## 📈 Key Insights
- Netflix has significantly more Movies than TV shows.
- The United States has the highest number of titles followed by India.
- Netflix experienced a major content releases around the year 2018.
- International movies and Documentaries are the most common genres.
- Most Netflix content has the duration of around 90-110 minutes.

## ✅ Conclusion
The project explored the Netflix Movies and TV shows using Exploratory Data Analysis. Through Data Cleaning and Data visualisation important insights about the Netflix's content were identified.
