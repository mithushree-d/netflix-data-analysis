# 🎥 Netflix TV shows and Movies

## 📌 Project overview
This project performs a exploratory data analysis(EDA) on the datset Netflix TV shows and Movies dataset using Python. The objective of this project is to analyze the netflix's content and discover meaningful insights about the movies, TV shows, genres, ratings, release year and many more using data visualization.


## 📂 Dataset
**Source : **Kaggle
**File name : **netflix_titles.csv

### Dataset Description
The datset contains the meaningful information about the netflix content such as movies and TV shows. It includes details such as :

- show_id
- type
- title
- director
- cast
- country
- date_added
- release_year
- rating
- duration
- listed_in (Genre)
- description


## 🚀 Technologies used

* Python - Prgramming language
* Pandas
* Matplotlib
* Seaborn
* VS Code - Code editor

## 📊 Exploratory Data Analysis

The following analysis were performed during EDA:
- Movies VS TV shows distribution
- Ratings distribution
- Release year distribution
- Top 10 Countries by content
- Movie Duration
- Top 10 Directors
- Top Genres
